# 微信注册
---------------------------------------------------------------------------
乐书急需要开发微信注册用户功能，这是一种有效且安全的注册方式。而且可以完整整个快速沟通流程。对乐书网活动推广
和业务扩展有积极作用。
所谓微信注册指，用户通过关注“乐书假日生活”微信，同时选择绑定用户，在乐书网创建新的用户，此时可认定为该用会为
有效的真实用户，可以很好的过滤垃圾用户和机器人，同时关注“乐书假日生活”微信以后，可以方便的通过微信查询活动，
乐书网也可以通过微信和每一位乐书用户保持沟通。
